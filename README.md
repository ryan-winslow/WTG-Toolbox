# WTG Toolbox

WTG Toolbox is a lightweight desktop utility app for common IT tasks such as network checks, system information, and file operations.

## Features

- Browse tools by category
- Run scripts from the desktop drawer
- View README and About information
- Keep the UI responsive while scripts run in the background

## Running the app

From the repository folder, run:

```bash
python wtg_toolbox.py
```

## Requirements

- Python 3.8+
- Tkinter support for your Python installation
